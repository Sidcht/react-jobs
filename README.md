# React Jobs

A modern job listing application built with React and Vite.

## Features

- Browse job listings
- View job details
- Add new jobs
- Edit existing jobs
- Delete jobs
- Responsive UI
- Toast notifications

## Built With

- React
- Vite
- React Router
- Tailwind CSS
- JSON Server
- React Icons
- React Toastify
