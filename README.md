# React Jobs

A modern job listing application built with React and Vite.

## Features

- Browse job listings
- View job details
- Add new jobs
- Read jobs
- Update existing jobs
- Delete jobs
- Responsive UI
- Persistent cloud database with Supabase

## Built With

- React
- Vite
- React Router
- Tailwind CSS
- Supabase
- React Icons
